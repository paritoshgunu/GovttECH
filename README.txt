1.Install Python3. 
2.Upgrade pip if you get it related pip  .
    Run this cmd " - python -m pip install --upgrade pip" in python path.
3.Install Selenium
    cmd - pip install selenium
4.Install requests
 cmd  pip install requests

5.Select the correct chrome driver based on your  browser version.
6.Download the "new_hero3.csv" file.
7.Update your csv file path in the python file.

8.Goto command prompt.
9.Run java -jar oppenheimer.jar in a cmd terminal
10.go to localhost:8080 and check page is loaded correctly.
11.Run Govtech_Auto.py file.  python Govtech_Auto.py
12.Again go to localhost:8080,and check all inserted records are displayed in the oppenheimer page automatically.
